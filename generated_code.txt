Request: Create a function that calculates the area of a rectangle given its length and width.

Function Name: calculateRectangleArea

Parameters: 
- length (number): length of the rectangle
- width (number): width of the rectangle

Return Type: number (representing the area of the rectangle)

Code:

```
function calculateRectangleArea(length, width) {
    //check if parameters are numbers
    if (typeof length !== 'number' || typeof width !== 'number') {
        return 'Error: Parameters must be numbers';
    }
    //calculate area by multiplying length and width
    let area = length * width;
    //return the area
    return area;
}
```
