# NEXT-19 pack_files (распакованный пакет)

Эта папка предназначена для **распакованных** файлов пакета, если загрузка ZIP через Make.com нестабильна.

## Рекомендуемая структура (когда появится код/скелеты)
- `contracts/` — схемы и контракты (JSON Schema, OpenAPI)
- `service/` — код/скелет сервиса (например FastAPI/Node)
- `projections/` — материализованные представления/индексы
- `examples/` — примеры payload/queries
- `docs/` — дополнительные документы
