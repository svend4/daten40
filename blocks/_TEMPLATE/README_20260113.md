# Шаблон блока IFOS (APPEND-ONLY)

Эта папка — **шаблон**, который копируется для каждого нового блока.

## Как пользоваться
1) Создайте новую папку блока: `blocks/NEXT-XX/` или `blocks/IFOS_#####_#####/`
2) Скопируйте/адаптируйте файлы из `tools/templates/` и сохраните их с датой:
   - `README_YYYYMMDD.md`
   - `spec_YYYYMMDD.md`
   - `manifest_YYYYMMDD.json`
   - `checksums_YYYYMMDD.json`
3) Если есть содержимое «пакета», публикуйте распакованные файлы в:
   - `pack_files_YYYYMMDD/` (папка)
4) (Опционально) Соберите ZIP локально скриптом `tools/build_pack.py` и положите в `releases/`.

## Почему так
Вариант 2 (append-only) избегает проблем Contents API (sha) и проблем с большими base64 payload.
