# NEXT-22 pack_files

Эта папка предназначена для **распакованных** файлов (в будущем), чтобы репозиторий мог работать без зависимости от больших архивов.

## Планируемая структура
- `contracts/` — контракты данных и спецификации API (schema/openapi)
- `service/` — скелет сервиса реестра исключений
- `workflows/` — шаблоны процессов согласования (approval)
- `examples/` — примеры запросов/выдач/контрактов
- `docs/` — эксплуатация, аудит, FAQ
