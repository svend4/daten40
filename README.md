## Что такое IFOS

**IFOS (Internet Function OS)** — B2B-платформа для каталогизации и стандартизации интернет-функций.  
Концепция: вместо изобретения новых инструментов — рационализировать и каталогизировать существующие в единый реестр с единым API-слоем.

Архитектура состоит из 6 уровней:
1. **Function Registry** — единый реестр всех интернет-функций
2. **Vocabulary / Schema** — унифицированные схемы данных
3. **Composer / Macros** — составные функции и макросы
4. **Blueprints** — готовые сценарии развёртывания
5. **Trust Layer** — репутация, SLA, аудит
6. **AI Layer** — интеллектуальная оркестрация

## OS_PACK формат

Каждый блок = пакет файлов:
- `IFOS_NNNNN_NNNNN_name.md` — основное описание функции
- `*_schema_v1.json` — JSON-схемы
- `*_api_v1.yaml` — OpenAPI спецификация
- `*_stub_v1.py` — Python-заглушка
- `*_examples_v1.json` — примеры использования

Нумерация блоков: **14241 → 88800** (текущее покрытие)
